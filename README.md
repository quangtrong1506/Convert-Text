# Convert Text [QT]

#### Shortcut

##### Open select Feature

-   Win: ctrl+alt+S,
-   mac: shift+cmd+S

##### Translate

-   Win: ctrl+alt+L,
-   mac: shift+cmd+L

##### Format

-   Win: ctrl+alt+F,
-   mac: shift+cmd+F

##### Convert

-   Win: ctrl+alt+C,
-   mac: shift+cmd+C

##### HTML To JSX

-   Win: ctrl+alt+X,
-   mac: shift+cmd+X
